# Memory-Flesh
Memory Flesh es una plataforma para guardar recuerdos con imagen y texto. Funciona como diario privado o foro público: vos elegís la privacidad. Si es pública, un admin la modera y recibís notificaciones de aprobación o desaprobación. Incluye gestión de perfil para ver tus datos y una sección de consulta(Perfil)para explorar todas tus memorias privadas y públicas.
